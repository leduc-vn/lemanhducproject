# AI Face Detection Web

A web application that detects human faces from uploaded images or webcam in real-time.
## Demo

![AI Face Detection Demo]("C:\xampp\htdocs\face-recognition-web\public\9403a235-45b4-4581-9c4c-71376c3eb18b.png")
## Technologies

- Laravel (Backend Web)
- Python Flask (AI API)
- OpenCV (Face Detection)
- MySQL (Database)
- JavaScript (Webcam capture)

## Features

- Upload image and detect faces
- Webcam realtime face detection
- Save detection history
- Dashboard statistics

## System Architecture

User (Browser)
↓
Laravel Web App
↓
Flask AI API
↓
OpenCV Face Detection

## Installation

### 1. Clone project

git clone https://github.com/leduc-vn/lemanhducproject.git

### 2. Run Laravel

php artisan serve

### 3. Run Flask API

cd flask-ai  
python app.py

### 4. Open browser

http://127.0.0.1:8000
